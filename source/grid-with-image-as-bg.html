Goal

  • Flexbox grid where flex items have a background image
  • By using flexbox it is assured that items that fit in a row will have the same height without using javaScript
  • By using background-size: cover we make sure the image will always span to the full width of the copy content
  • Con:
  • No usage of image srcset to save on performance
  • Not as good for SEO as the image can't be given the proper values so a duplicate image property or meta tag has to be added.
  • Alternative approach can be found here with an image tag instead.

Test results

  • Animation of the background image will only happen if the user doesn't hover over the copy
  • Animation of the backround image with translate will also reflect down to the copy that it wraps
